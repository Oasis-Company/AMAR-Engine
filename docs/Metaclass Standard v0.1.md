Oasis Metaclass Standard (OMS)
Version: 0.1
Status: Draft Specification


OMS v0.1 defines:

The minimal structural requirements for a valid metaclass

The distinction dependency model

Stability declaration format

Lifecycle declaration format

Room projection contract

Cross-metaclass interaction contract

OMS v0.1 does NOT define:

Rendering implementation

Physics engine

Serialization format

Performance constraints

UI layer
